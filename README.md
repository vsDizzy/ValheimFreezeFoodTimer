# Valheim Freeze Food Timer

Permanently freezes the food duration timer, keeping your food buffs active indefinitely.

![Preview](assets/header.jpg)

## Features

- **Zero performance impact:** Optimized logic, no heavy update loops.
- **Plug & Play:** Zero configuration needed.
- **Client-side:** Safe to use in multiplayer.
